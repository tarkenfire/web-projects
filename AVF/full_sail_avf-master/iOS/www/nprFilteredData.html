NPR Data Demo

Index

The NPR offers a web-based RESTful API which allows for searching their web-based content. It includes filters for various things such as video, audio, and text-only content.

For the sake of this demo, we will be using the basic search feature to find articles by search terms, entered into the search box below. Only results with text will be displayed.

However, in this mashup, the article content will be different depending on the type of connection on the device. For instance, when on wifi, pictures will be shown along with the articles, whereas when it is on 1x/3g/4g, results will have text only.
